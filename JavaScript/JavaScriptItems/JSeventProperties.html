Event types

click: Occurs when a mouse button is clicked on an element.
dblclick: Occurs when a mouse button is double-clicked on an element.
mousedown: Occurs when a mouse button is pressed down on an element.
mouseup: Occurs when a mouse button is released on an element.
mousemove: Occurs when the mouse pointer is moved while it is over an element.
mouseover: Occurs when the mouse pointer enters the area of an element.
mouseout: Occurs when the mouse pointer leaves the area of an element.
keydown: Occurs when a key on the keyboard is pressed down.
keyup: Occurs when a key on the keyboard is released.
keypress: Occurs when a key on the keyboard is pressed and released.
focus: Occurs when an element receives focus (e.g., via tabbing or clicking).
blur: Occurs when an element loses focus.
submit: Occurs when a form is submitted.
change: Occurs when the value of an input element changes (e.g., in text fields, checkboxes, or select dropdowns).
load: Occurs when a resource and its dependent resources (e.g., images, scripts) have finished loading.
unload: Occurs when a page is being unloaded (e.g., when the user navigates away from the page or refreshes it).

//The second square will just be doing the "toggle()" function, since we don't have the removeEventListener()

//Here we are just linking everytime the user click on the div "square"

textContent

The textContent property in JavaScript is used to get or set the text content of an element in the DOM (Document Object Model). It represents the textual content of the specified node and all its descendants.

You can also set the text content of an element by assigning a value to the textContent property. When you assign a value to textContent, it replaces all existing content within the specified element with the new text content. For example:

const example document.getElementById("example") //Here i am getting the id of the tag "p"

innerHTML

The innerHTML property in JavaScript is used to get or set the HTML content (including tags and text) of an element. It allows you to dynamically manipulate the content of an element by accessing or modifying its HTML markup.

When you access the innerHTML property without assigning a value, it returns the HTML content of the specified element, including any child elements, tags, and text. For example:
