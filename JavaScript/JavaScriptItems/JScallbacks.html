Here you will Learn about sychrony and Asynchrony.

Synchrony is a term we used to talk about code instructions that executes ONE AFTER ANOTHER

in synchronous code, one line of code is executed only after the previous has finished.

Synchronous: In synchronous programming, tasks are executed one after the other in a sequential manner. Each task must wait for the previous one to complete before it can start. This means that if one task takes a long time to execute

Asynchronous does not wait for the previous instruction.

In asynchronous programming, tasks can be executed independently of one another. Instead of waiting for a task to finish before moving on to the next one, asynchronous code allows tasks to run concurrently.
