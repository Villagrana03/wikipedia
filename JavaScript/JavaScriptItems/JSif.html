What's if?

if statement is a control flow statement that allows you to execute a block of code conditionally

let num = 10;

if(num < 10){
console.log("the numbers is less than 10")
else if(num > 10){
console.log("the number is more than 10")
}
}

if: what does if do is to make a condition, if the condition is met it will run a function that are inside of the {}

Else if: else if is similary to the if, but we only are going to use else if to make a second condition, for example, if the first condition is not met, with the else if we can have a second condition. we may have as many as we want.

Else: the Else statement is used when we do not require any condition.
