Classes

a class is a "template for objects" or a blueprint or template that defines the properties and behaviors of objects. Objects are instances of classes, created based on the specifications defined in the class.

We would use classes intead of objects because it is faster.

Here i will show how is a class made, first we start with the word "class" then with the superclass which in this case will be "Book" always the first letter uppercase

to create new objects, classes need a special method called constructor() thius method sets the property values for a new object, in the example i will have 2 parameters, that will specified unique values of the new object.

The purpose of the constructor is to set up the initial state of the object by assigning values to its properties or performing any other necessary setup tasks.

class Book {

constructor(author, tittle){

}

}

Before creating a new object property, we would need the special keyboard "this" we use this to refer the object being created, then we add a . and the property name

to finish creating the object property we will assign the author, as a VALUE to this.author

The line this.title = title; inside the constructor is assigning the value of the title parameter to the title property of the Book object being created. This line ensures that each Book object created with the Book class will have a title property set to the value provided when the object is instantiated.

// this is the class

class Book {

constructor(author, tittle){

this.author = author;

this.title = title;

}

}

const book1 = new Book()

Time to use our class, we start the same way we'd create a constant, but then add the "new" keyboard followed by Book()
Here we are assigning new Book() to the book1 constant

Then we will add values we want to set for the new object inside the parentheses

class Book {

constructor(author, tittle){

this.author = author;

this.title = title;

}

}

const book1 = new Book( "Suzanne Collins" , "hunger Games" )

const book2 = new Book( "George Lucas" , "star wars" )

console.log(book1) // output author: "Suzanne Collins", tittle: "Hunger Games"

console.log(book2) // output author: "George Lucas", tittle: "star wars"
