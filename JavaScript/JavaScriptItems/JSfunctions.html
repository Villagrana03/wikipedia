What is a function?

A function is a reusable block of code that performs a specific task or a set of instructions. It allows you to encapsulate a piece of code into a named entity, which can be called and executed multiple times throughout your program.

let a = 5;


let b = 4;

function sum (a,b){

// i would use normal functions instead of arrow when i want to use the keyword "this"

let sum = a + b;
return sum;
}

  1. The function is named addNumbers.
  2. It takes two parameters a and b, which represent the numbers to be added.
  3. Inside the function, the code calculates the sum of the two numbers and assigns it to the variable sum.
  4. The return statement is used to specify the value that the function will output. In this case, it returns the calculated sum.
