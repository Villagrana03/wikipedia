For Loop

A for loop is a programming construct used for repetitive execution of a block of code.

for( let i = 0; i < array.length; i++)

for(Initialization; Condition; Iteration)

For loop Parameters

the for loop takes 3 parameters, which are for(initialization; condition; iteration)

  1. Initialization: The initialization parameter is executed before the loop begins. It is used to initialize the loop variable or any other necessary variables. This parameter typically assigns an initial value to the loop variable.
  2. Condition: The condition parameter is evaluated before each iteration of the loop. If the condition is true, the loop continues executing. If the condition is false, the loop terminates
  3. Iteration: The iteration parameter is executed at the end of each iteration, just before evaluating the condition again. It typically modifies the loop variable

For In

The for...in statement iterates over all enumerable string properties of an object (ignoring properties keyed by symbols), including inherited enumerable properties.

// i have a great example inside "script"

const person = {
name: "Michelle",
age: 19,
city: 'Tijuana',
food: "Enchiladas" };

for (let key in person) { // This loop will only log properties directly defined on the person object and not properties inherited from its prototype chain.
console.log(key + ': ' + person[key]);
}

output // name: Michelle age: 19 city: tijuana food: enchiladas

for Of

The for...of loop in JavaScript is used to iterate over iterable objects such as arrays, strings, sets, maps, and other data structures that implement the iterable protocol. It provides a simpler and more concise syntax compared to traditional for loops or for...in loops, especially when working with arrays.

// in the for of, i gotta use 2 parameters, if i only use 1, the result would be "4" in this case. If a instead of returing value, i would return index (first parameter) it would only return the index of each item, like in the second example

const array = ['hello', 'world', 'this', 'is', 'great'];
let result = array.entries() // Using entries() to iterate through the array
const prueba = () => {
let resultado = ""
for(const [index, value] of result){ // Using a for...of loop to iterate through the iterator
resultado += value + " "
}
return resultado
}
console.log(prueba())
// second example

output // hello world this is great
