Dictionaries

Dictionaries are a built-in data type in Python used to store collections of key-value pairs. They are unordered, mutable, and indexed by keys, which can be of any immutable type (such as strings, numbers, or tuples).

Dictionaries

Unordered: Prior to Python 3.7, dictionaries were unordered collections. From Python 3.7 onwards, they maintain insertion order.
Indexed by Keys: Accessed by using keys rather than numerical indexes.
Mutable: You can change, add, or remove key-value pairs after the dictionary is created.
Unique Keys: Keys must be unique within a dictionary

traduction = {

Accessing Dictionary Elements

You can access the values in a dictionary by using their keys.

traduction = {
"Blue": "Azul",
"Green": "Verde",
"Red": "Rojo":,
"Yellow": "Amarillo"
}


print(traduction["Azul"]) Output -> "Blue"

Modifying Dictionaries

traduction = {
"Blue": "Azul",
"Green": "Verde",
"Red": "Rojo",
"Yellow": "Amarillo"
}

# Adding a new key-value pair
traduction["Purple"] = "Morado"
print(traduction) Output -> {'Blue': 'Azul', 'Green': 'Verde', 'Red': 'Rojo', 'Yellow': 'Amarillo', 'Purple': 'Morado'}

# Updating an existing key-value pair
traduction["Blue"] = "Mango"
print(traduction["Blue"]) Output -> "Mango"

# Deleting a key-value pair
del traduction["Blue"]
print( traduction) Output -> {'Green': 'Verde', 'Red': 'Rojo', 'Yellow': 'Amarillo'} # ""Blue was deleted

traduction.pop("Blue")
print(traduction) Output -> {'Green': 'Verde', 'Red': 'Rojo', 'Yellow': 'Amarillo'} # ""Blue was deleted

Dictionary Methods

Python provides a variety of methods to work with Dictionary:

Removes all elements from the dictionary.

traduction = {
"Blue": "Azul",
"Green": "Verde",
"Red": "Rojo",
"Yellow": "Amarillo"
}

traduction.clear()

setdefault()

Returns the value of the specified key. If the key does not exist, inserts the key with the specified value.

traduction = {
"Blue": "Azul",
"Green": "Verde",
"Red": "Rojo",
"Yellow": "Amarillo"
}

traduction.setdefault("newValue", "Hello!")
print(traduction) Output -> {'Blue': 'Azul', 'Green': 'Verde', 'Red': 'Rojo', 'Yellow': 'Amarillo', 'newValue': 'Hello!' } #add a value to the the dictionary

update()

Updates the dictionary with elements from another dictionary or an iterable of key-value pairs.

myDict = {
"a": 1,
"b": 2
}

myDict.update({"b": 54, "c": 3})
print(myDict) Output -> {'a': 1, 'b': 54, 'c': 3} # with update, we can update a value of a key to anything OR add new keys to the dictionary

values()

Returns a view object that displays a list of all the values in the dictionary.

traduction = {
"Blue": "Azul",
"Green": "Verde",
"Red": "Rojo",
"Yellow": "Amarillo"
}

values = traductionvalues()
print(values) Output -> dict_values(['Azul', 'Verde', 'Rojo', 'Amarillo']) # gets just the values of the dictionary

items()

Returns a view object that displays a list of a dictionary's key-value tuple pairs.

traduction = {
"Blue": "Azul",
"Green": "Verde",
"Red": "Rojo",
"Yellow": "Amarillo"
}

tuppleExample = traduction.items()
print(tuppleExample) Output -> dict_items([('Blue', 'Azul'), ('Green', 'Verde'), ('Red', 'Rojo'), ('Yellow', 'Amarillo')]) #get every pair of the dictionary and makes every pair a tupple

keys()

Returns a view object that displays a list of all the keys in the dictionary.

traduction = {
"Blue": "Azul",
"Green": "Verde",
"Red": "Rojo",
"Yellow": "Amarillo"
}

listExample = traduction.keys()
print(listExample) Output -> dict_keys(['Blue', 'Green', 'Red', 'Yellow']) #get every key value of the dictionary and put them into a list

fromkeys()

Creates a new dictionary with keys from an iterable and values set to a specified value.

letters = ["a", "b", "c"]
defaultValue = 0

newDictionary = dict.fromkeys(letters, defaultValue)
print (newDictionary) Output -> {'a': 0, 'b': 0, 'c': 0} # What is it doing here, is getting a list and putting a default value and converting that list into a dictionary
