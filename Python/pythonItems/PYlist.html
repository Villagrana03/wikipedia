Python List

In Python, a list is a built-in data structure that allows you to store an ordered collection of items. Lists can contain elements of different types (integers, floats, strings, other lists, etc.), and they are mutable, meaning you can change their content after creation.

Ordered: Elements have a defined order, and this order will not change unless explicitly modified.
Indexed: Each element in a list has an index, starting from 0.
Mutable: You can change elements, add new elements, and remove elements after the list has been created.
Allows Duplicates: Lists can have multiple occurrences of the same element.

List Methods

Python provides a variety of methods to work with lists:

Append()

Append adds a value at the end

numbers = [1, 2, 3, 4, 5]
numbers.append(9)

print(numbers) Output -> [1, 2, 3, 4, 5, 9]

extend()

Adds all elements of an iterable (list, tuple, set, etc.) to the end of the list.

numbers = [1, 2, 3, 4, 5]
numbers.extend([11, 12, 13]) // in order to use many values must be inside of a list in the extend

print(numbers) Output -> [1, 2, 3, 4, 5, 11, 12, 13]

insert()

Inserts an item at the specified position. insert(index, item)

numbers = [1, 2, 3, 4, 5]
numbers.insert(1, 99) // 1 is the index and 99 the number that would be introduced at index 1

print(numbers) Output -> [1, 99, 2, 3, 4, 5]

pop()

Removes and returns the element at the specified position (default is the last element)

numbers = [1, 2, 3, 4, 5]
numbers.pop()

print(numbers) Output -> [1, 2, 3, 4] # since we did not declare anything inside of pop. we will remove the last item.

numbers.pop(1)

print(numbers) Output -> [1, 3, 4, 5] # In this example we have 1 inside of pop, which means that the index 1 at numbers will be deleted.

remove()

Removes the first occurrence of an item.

numbers = [1, 2, 3, 4, 5, 3]
numbers.remove(3)

print(numbers) Output -> [1, 2, 4, 5, 3] // it only removes the first value that you give, for example it removed the first 3

pop(index)

Removes and returns the item at the specified position (default is the last item).

numbers = [1, 2, 3, 4, 5]
numbers.pop(3)

print(numbers) Output -> [1, 2, 3, 5] // it deletes the index 3 element

clear()

Removes all items from the list.

numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) Output -> []

index()

Returns the index of the first occurrence of an item

numbers = [1, 2, 3, 4, 5]

print(numbers.index(2)) Output -> [3]

count()

Returns the number of occurrences of an item.

numbers = [1, 2, 3, 4, 5, 4, 4, 4, 4 ,4]

print(numbers.count(4)) Output -> 6 // it counts the number of 4

reverse()

Reverses the order of the list.

numbers = [1, 2, 3, 4, 5]
numbers.reverse()

print(numbers) Output -> [5, 4, 3, 2, 1]

sort()

Sorts the list in ascending order (can be modified with parameters).

numbers = [3, 5, 1, 4, 2]
numbers.sort()

print(numbers) Output -> [1, 2, 3, 4, 5]

copy()

Returns a shallow copy of the list.

numbers = [1, 2, 3, 4, 5]
numbersCopy = numbers.copy()

print(numbersCopy) Output -> [1, 2, 3, 4, 5]
