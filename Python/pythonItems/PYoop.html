OOP Programming

Python classes are a way to bundle data and functionality together. Creating a new class creates a new type of object, allowing new instances of that type to be made. Each instance can have attributes attached to it for maintaining its state. Python classes provide a means of bundling data and functionality together

Basic concepts Concepts of OOP

A class is a blueprint for creating objects (a particular data structure). A class defines a set of attributes and methods that the created objects will have.

Class Attribute: A variable that is shared among all instances of a class.
Instance Attribute: A variable that is unique to each instance of a class.
Instance Method: A function that operates on an instance of a class and can access instance attributes.

# A class is defined using the class keyword.:
class Animal:

# Class attribute
alive = True

# Initializer / Instance attributes
---def __init__(self, name, age):
------ self.value = value
------ self.name = name
------ self.age = age

---def presentation(self):
------ return (f"Hello my name is {self.name} and i am {self.age} old!")

Inheritance in Python

Inheritance is a feature in object-oriented programming that allows a class (called a child or subclass) to inherit attributes and methods from another class (called a parent or superclass). This promotes code reuse and can create a hierarchical relationship between classes.

Key Points

Parent Class (SuperClass): The class whose attributes and methods are inherited.
Child Class (SubClass): The class that inherits from the parent class.
super(): A function used to call methods from the parent class.

class Cat(Animal):

#The parent class have name and age instance attributes, but Cat has additional attributes, such as "mustache"
def __init__(self, name, age, mustache):
--- #The super() method is used to bring all the instance attributes from the parent to the child, and then we proceed to create unique attributes for the child
--- super().__init__(name, age)
--- self.mustache = mustache #in this line we created Cat attribute.


def mustacheCount(self):
return (f"Since i am a cat i have {self.mustache} mustaches!")

# Creating Objects
azul = Cat("Azul", 4, 13) #calling Cat method print(azul.mustacheCount())
