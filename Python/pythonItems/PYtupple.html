Tupple

Tuples are a type of data structure in Python that are similar to lists but Immutability: Once created, the elements of a tuple cannot be changed (unlike lists which are mutable).

Tupple

coords = (54, 21)
data = ("Michelle", 19, True) # Tuple with mixed data types

print(coords) Output -> (54, 21) print(data) Output -> ("Michelle", 19, True)

Tupple Methods

Since tuples are immutable, they have fewer methods compared to lists. Here are the methods available for tuples:

count()

Returns the number of times the specified item appears in the tuple.

tupple = (1, 2, 3, 4, 5, 4, 4, 4, 4, 4)

print(tupple.count(4)) -> Output = 6

index

Returns the index of the first occurrence of the specified item.

tupple = (1, 2, 3, 4, 5)

print(tupple.index(2)) -> Output = 3
