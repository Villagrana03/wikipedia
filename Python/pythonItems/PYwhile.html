While Loop

In Python, a while loop is used to repeatedly execute a block of code as long as a specified condition is true. This means the While loop would be iterating as long is the condition is true, if its false the loop will be broken

While loops

This is an example of normal while loops, i will be explaining in the comments how they work.

The condition is evaluated. If the condition is True, the code block inside the loop is executed. If the condition is False initially, the code block is skipped entirely, and the program continues with the statement immediately following the while loop.

secretWord = "mango"
userWord = ""
lives = 3

while(userWord != secretWord and lives != 0):
// Here i set userWord is not equal to secretWord, Here is an example, is "mango" not equal to "azul" (pretedning "mango" is the secretWord and "azul" userWord) are those words equal True or false? the answer is True because "mango" is not eqaul to "azul" so the loop will iterate again until one of the condition is false.
userWord = input("Enter a word: ")
lives -= 1
print(f"You have {lives} remaining and the word you tried is {userWord}")
