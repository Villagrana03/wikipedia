Set

In Python, sets are used to store multiple items in a single variable. They are one of the built-in data types in Python and have several useful properties and methods. Here are some key points about sets: ,use {} same same as dictionaries


Why would we use sets?

Sets are particularly useful when you need to eliminate duplicate entries from a collection or when you need to perform mathematical set operations like union, intersection, difference, and symmetric difference.

Unordered: set does not mantain order of the data

Unindexed: Items in a set do not have an index, so you cannot access elements by an index.

Unique Elements: A set cannot contain duplicate elements.

Mutable: Sets are mutable, meaning you can add and remove items from a set.

Creating Sets

You can create a set using curly braces {} or the set() constructor:

# Using curly braces
my_set = {1, 2, 3, 4}

# Using the set() constructor
my_set = set([1, 2, 3, 4])

Set methods

Here i will show set methods

add()

add a value to the set

conjuntosA = {1, 3, 5, 7, 8, 3, 3, 1}
conjuntosB = {2, 4, 5, 6, 8, 2}

conjuntosA.add(9) Output -> {1, 3, 5, 7, 8, 9} # Remember that with sets all the duplicates are being removed.

remove()

remove a value of the set, Raises a KeyError if the element is not present.

conjuntosA = {1, 2, 3}
conjuntosB = {3, 4, 5}

conjuntosA.remove(2) Output -> {1, 3} # Remember that with sets all the duplicates are being removed.

clear()

Removes all elements from the set.

conjuntosA = {1, 2, 3}
conjuntosB = {3, 4, 5}

conjuntosA.clear() Output -> {}

Union()

Join both sets

conjuntosA = {1, 3, 5, 7, 8, 3, 3, 1}
conjuntosB = {2, 4, 5, 6, 8, 2}

conjuntoC = conjuntosA.union(conjuntosB)
print(conjuntoC) Output -> {1, 2, 3, 4, 5, 6, 7, 8} #Union joins both sets, removing all the duplicates and ordering in ascending order.

intersection()

Gives you the values are in both set

conjuntosA = {1, 3, 5, 7, 8, 3, 3, 1}
conjuntosB = {2, 4, 5, 6, 8, 2}

conjuntoC = conjuntosA.intersection(conjuntosB)
print( conjuntoC) Output => {5, 8} # give the number that has both set

difference()

difference of the set

conjuntosA = {1, 2, 3}
conjuntosB = {3, 4, 5}


conjuntoC = conjuntosA.difference(conjuntosB)
print( conjuntoC) Output => {1, 2} # give the number unique in ConjuntoSA (numbers that are not shared in the other set)

conjuntoC = conjuntosB.difference(conjuntosA)
print( conjuntoC) Output => {4, 5} # give the number unique in ConjuntosB (numbers that are not shared in the other set)
